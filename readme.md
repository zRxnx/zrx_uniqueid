# zRxnx's Advanced Unique Identifier System

An advanced unique identifier system for FiveM

**https://docs.zrxnx.at**

**https://discord.gg/mcN25FJ33K**

## About

### Features

- Menu to edit existing UIDs
- Search feature
- ox_lib log
- Update checker
- Optimized
- Synced
- 0.0 ms on idle
- 0.0 ms while in use

### API

#### Exports

- GetPlayerUIDfromSID (replicated)
- GetPlayerSIDfromUID (replicated)
- ChangePlayerUID (server)

#### Statebags

- zrx_uniqueid:uid

## Requirements

- ox_lib (latest)
- oxmysql (latest)

## Installation

- Download script from Releases
- Put script in your `resources` directory
- Add `ensure zrx_uniqueid` in your `server.cfg`
